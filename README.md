# Uncle App

Website: https://uncleapp.carrd.co/

**Uncle App** is an iOS application designed to help individuals set measurable goals, build structured routines, and develop productive digital habits. The app leverages Appleâ€™s Family Controls APIs to support goal-setting in a safe, transparent, and consent-based way.

---

## Features

- **Goal Setting:** Create personal or family goals with clear milestones.
- **Progress Tracking:** Monitor progress over time and measure achievements.
- **Structured Habits:** Build routines that encourage focus and productivity.
- **Parental/Guardian Support:** Guardians can guide goal achievement with consent.
- **Data-Driven Insights:** Provides actionable insights on habits without invasive tracking.

---

## Technology Stack

- **Language:** Swift / SwiftUI
- **APIs:** Family Controls (DeviceActivity, ManagedSettings)
- **Minimum iOS Version:** 16.0+
- **Architecture:** MVVM

---

## Getting Started

### Prerequisites

- macOS with Xcode 15+
- Paid Apple Developer account (required for Family Controls entitlements)
- iPhone running iOS 16+ for real device testing
